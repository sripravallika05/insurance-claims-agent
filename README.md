# Autonomous Insurance Claims Processing Agent

AI-powered backend system for insurance claim PDF processing.

## Features

- PDF upload
- Text extraction
- Field extraction
- Validation
- Routing engine
- FastAPI backend

## Tech Stack

- FastAPI
- Python
- pdfplumber
- Regex
- Uvicorn

## Run Project

pip install -r requirements.txt

python -m uvicorn app.main:app --reload
